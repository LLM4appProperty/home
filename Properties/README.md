## Property descriptions used in out evaluation

You can find the 124 property descriptions in `original_properties.xlsx`:

* ID: the identifier for each property description.
* App: the app name of the property description.
* Property Description: the property description in natural language.

You can find the paraphrased property descriptions in `paraphrased_properties_{gpt4o or deepseekv3}.xlsx`

You can find the 36 original and corresponding paraphrased property descriptions from new 12 app subjects in `extended_subjects_original_properties.xlsx`, `extended_subjects_paraphrased_property_deepseek.xlsx` and `extended_subjects_paraphrased_property_gpt4o.xlsx`
